### 웹디자인 기능사 레이아웃 연습
반응형 레이아웃을 연습하고 웹디자인기능사 레이아웃 유형과 스크립트의 메뉴 유형, 슬라이드 유형, 탭 메뉴 유형, 팝업 유형을 연습했습니다.
자바스크립트와 제이쿼리를 통해 스크립트를 작성했습니다.
각 유형을 학습 후 실전 사이트 5개 유형을 만들어봤습니다.
<br>
<br>
### 현대백화점 포인트 클론 사이트
피그마(Figma)와 제플린(Zeplin)으로 디자인 작업하고 웹 표준 코딩을 완성한 현대백화점 포인트 클론 사이트입니다.
검색 엔진 최적화를 위해 description, keywords, author, robots 등의 메타 태그를 활용했습니다.
또한 role과 aria-label을 사용해 웹접근성을 높였습니다.
이미지 슬라이드 기능을 구현하기 위해 Swiper 라이브러리를 사용했습니다. Swiper는 터치 슬라이드 지원 및 자동 재생과 같은 다양한 기능을 제공합니다.
반응형으로 디자인해, 화면 너비가 1200px 이하일 때 네비게이션 메뉴를 숨기고 햄버거 메뉴를 보이도록 했습니다.
<br>
<br>
### 웹표준 사이트 만들기
피그마(Figma)로 디자인을 완성하고 HTML과 CSS를 사용하여 웹페이지를 구현했습니다.
헤더 유형, 슬라이드 유형, 이미지 유형, 이미지/텍스트 유형, 카드 유형, 배너 유형, 텍스트 유형, 푸터 유형을 학습했습니다.
